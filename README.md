


KULOG - 일본 여행지 맛집 추천, 리뷰 게시판 (CRUD SERVICE)

### 주요기능 ###
- 지역 카테고리 (훗카이도, 관동, 간사이, 규슈) 별 따로 게시판 기능 구현
- 사진 슬라이드, 미리보기 
- 게시글, 댓글 , 저장 (사진파일 업로드)
- 회원가입, 로그인 (소셜로그인 포함)
- 북마크 기능 (추가, 조회, 삭제)



1. 기본 게시판 기능
- 각 지역별 게시판 따로 설정
- 게시글 작성, 파일 업로드, 수정 기능 구현
- 댓글 작성 (익명도 허용)












### 추가할 기능 ###
- 게시글 등록 시 지도 API 추가 
- 마이페이지, 프로필 화면
- 쪽지 기능
